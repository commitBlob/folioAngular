# folioAngular

Personal portfolio SPA of Maro Radovic, built with Angular 5 (Angular CLI 1.7.3). All content is served from static JSON files in `src/assets/data/` — there is no backend.

## Development

```bash
npm start            # Dev server on port 8888
npm run build        # Dev build → dist/
npm run build:prod   # Production build → dist/
npm test             # Jest unit tests
npm run test:watch   # Jest in watch mode
npm run lint         # TSLint
```
